PyQt5 Practice: Календарь событий

Описание проекта:
Данный проект представляет собой десктопное GUI-приложение «Календарь событий», разработанное в рамках учебной практики за 1 курс (автор: Краснов Вячеслав  группа: ФМ-14-25). 
Проект демонстрирует переход от консольного к событийному программированию с использованием фреймворка PyQt5[cite: 1]. 
Приложение реализует паттерн разделения логики и пользовательского интерфейса, а также выполняет операции CRUD (создание, чтение, обновление, удаление) с использованием локальной базы данных SQLite[cite: 1].

Запуск проекта:
1. Создайте виртуальное окружение: `python -m venv venv`[cite: 1]
2. Активируйте:
   - Windows: `venv\Scripts\activate`[cite: 1]
   - macOS/Linux: `source venv/bin/activate`[cite: 1]
3. Установите зависимости: `pip install -r requirements.txt`[cite: 1]
4. Запустите: `python main.py`[cite: 1]

Структура проекта:
- `main.py` → Точка входа, настройка QApplication[cite: 1]
- `ui_main.py` → Интерфейс, сигналы, обработка событий[cite: 1]
- `database.py` → Работа с SQLite (CRUD)[cite: 1]
- `requirements.txt` → Список зависимостей (PyQt5, Pillow)[cite: 1]
- `.gitignore` → Исключения для Git (venv, __pycache__, *.db)[cite: 1]
- `assets/` → Ресурсы (изображения, иконки, если используются)[cite: 1]
