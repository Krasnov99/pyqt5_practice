# Календарь событий

Десктопное GUI-приложение для управления расписанием. Проект демонстрирует применение паттерна разделения логики и пользовательского интерфейса.

## Технологии
* Python 3
* PyQt5 (графический интерфейс)
* SQLite (локальная база данных)

* Pillow (работа с изображениями)

## Реализованный функционал
* Добавление, редактирование и удаление запланированных событий.
* Проверка на пересечение времени задач.
* Цветовая маркировка различных типов мероприятий.
* Загрузка и отображение фотографий.

1. Установите необходимые зависимости из файла:
   `pip install -r requirements.txt`
2. Запустите главный файл проекта:
   `python main.py`
